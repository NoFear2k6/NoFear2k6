Hướng dẫn chi tiết về Cách Def Rally

Cập nhật mới nhất: 12/10/2025

Khi mà def rally, bạn nên làm những việc như sau:

  • Kiểm tra tướng thủ thành, kích hoạt hiệu ứng cuồng chiến (nếu cần)
  • Kiểm tra kẻ địch: đồ chiến, lính, ....
  • Khi def rally, tuỳ vào lính của kẻ địch để quyết định trận hình sẽ def
  • Nếu địch quá mạnh, hãy ưu tiên khiên hoặc di trú
